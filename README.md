# Ziya & Mədinə

Ziya və Mədinənin özəl bağı üçün hazırlanmış interaktiv, sirli xatirə saytı.

## Açmaq

`index.html` əsas saytıdır. Giriş kodu: `2610`.

`gallery.html` ayrıca qorunan qalereyadır. Giriş kodu: `2006`.

`sual.html` gizli sual və sürpriz səhifəsidir. Daimi şəkillər və daha şəxsi mətnlər növbəti mərhələdə əlavə ediləcək.

`music.html` ətraflı mahnı siyahısı, sözlər, seçilən misra və mahnı hekayələri üçün hazırlanmış musiqi otağıdır.
